Build a production-ready Windows desktop application called “Windows Action Assistant”.

MISSION

Create a lightweight Windows assistant that helps users perform actions, solve PC problems, launch projects, open applications, navigate Windows settings, and access information using natural language.

The application is NOT a chatbot and NOT a general AI assistant.

The application’s purpose is to translate user goals and problems into Windows actions.

Examples:

* “My screen is too bright”
* “Turn on hotspot”
* “Open startup apps”
* “Install Chrome”
* “Open my Outlook agent project”
* “I can’t hear anything”

CORE PRINCIPLE

Users should never need to know Windows terminology.

Users describe what they want.

The assistant finds the correct Windows action.

ARCHITECTURE

User Input
→ Intent Engine
→ Knowledge Base
→ Action Engine
→ Windows Action

Fallback:

→ Preferred Browser Search
OR
→ Preferred LLM
